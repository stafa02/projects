The code is complete and fully functional and was tested on remote.cs.binghamton.edu

steps to compile and run:

i) cd to directory madib1_java_p2
ii) $ make clean
It will cleanup previously compiled classes.

iii) $ make 
It will compile the java files present in madib1_java_p2 directory.

iv) $ java assignmentP2 <path-to-input-file>/<filename> <path-to-output-file>/<filename>
Run the program with two arguments, i.e, input and output file paths.


Java files included in foler:
i) assignmentP2.java 
Client for tsp, contains main 

ii) Tsp.java
contains branch and bound algorithm for tsp

iii) Node.java
Wrapper for object to be added to priority queue
