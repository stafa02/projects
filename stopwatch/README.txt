Files icluded:
i) stopwatch.html: Contains the UI for the assignment.
ii) stopwatch.js: JavaScript file containing implementation for stopwatch.
iii) driver.js: JavaScript file containing functions invoked by UI events.

Steps to run:
i) Open stopwatch.html in a browser.
ii) Enter time in seconds.
iii) Press update to update the new value.
iv) Use controls for each stopwatch to start/pause/reset.

Note:
Tested in chrome and firefox
