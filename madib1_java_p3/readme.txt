The code is complete and fully functional and was tested on remote.cs.binghamton.edu

steps to compile and run:

i) cd to directory madib1_java_p3
ii) $ make clean
It will cleanup previously compiled classes.

iii) $ make 
It will compile the java files present in madib1_java_p3 directory.

iv) $ java assignmentP3 <path-to-input-file>/<filename> <path-to-output-file>/<filename>
Run the program with two arguments, i.e, input and output file paths.


Java files included in folder:
i) assignmentP3.java
client for knapsack problem, contains main. 

ii) KnapsackSolution.java
contains dynamic programming algorithm for 0/1 knapsack problem.
